# Country / Process Documents (UI5 Mock)

## What you get
- Left: Tree (All -> Country -> Process) built from mock OData data
- Right: Table of documents (OData V2)
- Filters applied when you click a node; search field filters additionally

## Run in BAS
1. Open the folder `project1` in BAS
2. `npm install`
3. `npm run start`
4. The preview opens `webapp/index.html`

## Data
- OData V2 metadata: `webapp/localService/mainService/metadata.xml`
- Mock data: `webapp/mockdata/Documents.json`
